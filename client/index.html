Profile

Creates a separate dejavu folder for each document (e.g., "MyFile.ai/dejavu/").

Absolute path or "~/" for home. Per-document folders are relative to this location.

Locked tokens: $defaultFolder, $filename. Add date/time tokens between them.

— Preview

Use tokens to customize the dejavu filename format.

Recovery

Shows a warning when opening a document that has a newer dejavu.

Opens the latest dejavu for up to five documents, then creates a fresh recovery copy.

Autosave

Creates a backup of the original file before overwriting it with a new dejavu.

Automatically pins snapshots at regular intervals to prevent cleanup.

Prevents waiting a full interval after turning dejavu on.

Useful when open documents use different dejavu rules.

Updates

Periodically checks GitHub Releases and shows a banner when a newer version of DejaVu is available.

Save rules

When enabled, reuses the same filename instead of creating unique copies.

Skips dejavu if the document hasn't been modified since the last save.

Cleanup

Set 0 to disable a limit. Cleanup runs after each dejavu or from Timeline.

Cache health

Disk checks warn before storage gets tight and recovery limits keep unsaved sessions bounded.
